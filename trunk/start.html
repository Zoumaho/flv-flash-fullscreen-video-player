2. Folders

At first, I want to give you an overview of the folders in this download:

  • flvplayer: These are the main files of the flv player.
  • additional-assets: Here you can find some more skins.
  • src: Source files of the flv player (Flash files and ActionScript classes).
  • license: Information about the licenses
  • wordpress-plugin: A plug-in for the blog system WordPress
  • flash-cs3-demo: Example to how to use the player in actionscript

3. Deploying FLV Player on a webserver

You can upload the whole folder flvplayer on your server. If everything works, just delete the files you don't need.

At least, you need normally the following files on your live system:

  • flvplayer.swf and playbackengine.swf (main files of the player)
  • All files starting with "default..." (e.g. defaultpreloader.swf, defaultbuttonoverlay.swf, etc.). You can also replace these files with your own, if you want to have custom default assets.
  • flashvideoplayer.html if you use the link method to pass the parameters to the flv player.
  • swfobject.js for embedding flash in html and expressInstall.swf for the express install feature of swfobject
All the others assets are optional (e.g. alternative skins or preloaders, demos, demo content, etc.).
